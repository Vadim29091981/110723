
1.Асимптотический анализ - это…

Ответ: Time Complexity

2.Какой алгоритм быстрее?

Ответ: O(n)

3. Какой базовый случай для рекурсии будет работать, если n >= 30 && n < 100?

Ответ: int f(int n) if (n <= 1) return 1; else return n*f(n - 1)

4.Выберите анализ алгоритма, который оценивает алгоритм в лучшем случае.

Ответ: Big Omega notation

5.Верно ли утверждение: существует только один возможный алгоритм для конкретной задачи.

Ответ: Неверно

6.Если вы используете линейный поиск в массиве «Ar», какое значение будет наихудшим?

Ответ: 9

7.Выберите алгоритм O(n^2)?

Ответ: Сортировка пузырьком

8.Какая из следующих структур данных НЕ является линейной структурой данных?

Ответ: Деревья

9.Функция, которая находит местоположение элемента с заданным значением - это:

Ответ: Поиск

10.Верно ли утверждение: в красно-черном дереве у черного узла должны быть только красные потомки.

Ответ: Верно

11.Что из следующего верно для красно-черного дерева?

Ответ: Каждый лист (NULL) черный.

12.Стек - это…

Ответ: Last in, first out

13.Сколько итераций потребуется для сортировки следующего массива с использованием алгоритма пузырьковой сортировки?

Ответ: 4

14.Очередь (queue) НЕ поддерживает следующую операцию:

Ответ: Восстановление

15.Допустимые значения высоты дерева AVL:

Ответ: -1, 0, 1

16.Что из перечисленного является двойным вращением?

Ответ: LRL

17.Последовательность вставки 1, 2, 4 приведет к дисбалансу. Какое вращение можно использовать для балансировки дерева?

Ответ: LL

18.Какова временная сложность подсчета количества элементов в связанном списке?

Ответ: O(n)

19.Класс Связанного списка реализуется из…

Ответ: Interface List

20.Учитывая следующий метод, что вернет тест, если a = {0, 2, 3, 4} and v = 1?

Ответ: -1

21.Верно ли утверждение: алгоритм бинарного поиска можно использовать только в том случае, если данные упорядочены.

Ответ: Верно

22.Верно ли утверждение: пузырьковая сортировка — это эффективный алгоритм, реализующий подход «разделяй и властвуй».

Ответ: Неверно

23.Линейный поиск — это эффективный алгоритм, реализующий подход «разделяй и властвуй».

Ответ: Неверно
